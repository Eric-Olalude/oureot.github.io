07. Deploy to the Test Network

Deploy to the Test Network

To build and interact with decentralized applications you'll need a tool known as Metamask. Metamask provides the connection between DApps and the Ethereum blockchain.

Here we'll get you up and running with Metamask. Using this you'll deploy your DApp to the Ethereum Test Network. This will allow you to look up it's contract and transaction details using Etherscan.

Objectives

Our goal is to help you do the following things.

  1. Describe the importance of Metamask as a tool for Ethereum applications.
  2. Install and navigate Metamask
  3. Use a public faucet to get your first test Ethereum into your account.
  4. Deploy a contract to the Ethereum Test Network and investigate it's transactions using Etherscan.

1. Metamask Introduction

Metamask is an extension to the chrome and brave browsers. It allows you to interact with Ethereum applications. Understanding this tool is an extremely important part of becoming a blockchain developer.

The following video does an excellent job introducing Metamask. It was made by the creators of this tool and can also be found on their home page metamask.io or on YouTube.

2. Install Metamask

To interact with existing blockchain applications or build your own, you'll need to install the Metamask browser extension. Go to Metamask.io to learn more and install it on your browser.
