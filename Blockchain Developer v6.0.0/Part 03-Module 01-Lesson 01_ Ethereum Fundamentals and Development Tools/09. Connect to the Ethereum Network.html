09. Connect to the Ethereum Network

Connect to the Ethereum Network

Here you'll make your connection to the Ethereum main network. Using this connection, you'll be able to run basic commands to search the network from your command line.

This will be done using a tool known as Infura. By connecting web3 to Infura, you will be able to access the Ethereum main and test networks using the Infura API.

Objectives

Our goal is to help you do the following things.

  1. Define the importance of Infura and describe how it helps you develop DApps.
  2. Create an Infura account
  3. Establish a connection between web3 and an Ethereum network.
  4. Use your web3-infura connection to run basic commands that search the Ethereum network.

1. Introduction to Infura

In the following video I'll go over why Infura was created and the value it brings to your future blockchain applications.

9.1-Introduction-To-Infura

Reflect

QUESTION:

In 3 sentences or less, describe the main benefits of using Infura.

ANSWER:

Infura allows you to connect to the Ethereum blockchain without running a full node. It's a lightweight alternative to downloading the entire blockchain to your local device. It makes the connection that allows you to use the functionality provided by web3.

2. Up and Running with Infura

Now that you understand why infura such a useful tool, you can create an account to access the Ethereum network. To create an infura account, go to infura.io.

From there you can navigate to your dashboard and find the endpoints required to establish a connection to the blockchain. Below is a look at the Infura dashboard. If you can see this, you're all set!

Which endpoints are available for you to connect to from the Infura dashboard?

Step 2: Set a variable equal to a string of the address you chose.

var address = '0x8E21D1CE02fD6Cec455Da473b9D969424bAa9678'

Step 3: Use web3 to read the balance of this account and set it to the variable balance.

web3.eth.getBalance(address, (err, bal) => { balance = bal })

Visit the documentation for more information on web3.eth.getBalance.
