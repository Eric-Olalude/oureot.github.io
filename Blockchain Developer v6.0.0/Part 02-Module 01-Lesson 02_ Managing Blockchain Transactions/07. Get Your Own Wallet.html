07. Get Your Own Wallet

Bitcoin Wallet Overview

Wallets types and private keys are interesting, but so far it's all been theory. What we want you to do now is go out and get a wallet for yourself!

Getting a wallet and working with it for yourself helps you see what’s really going on with these new concepts. It also gives you the opportunity to create your very own blockchain identity!

Choose a Wallet

First, we have to pick a wallet.

There’s a lot of wallets to choose from, and any of them might make sense for you depending on your needs. Feel free to choose any wallet you like, but do some research before installing one for yourself.

A good way to get started is just to google ‘bitcoin wallets’ and start looking around.

Electrum

For this course, we’re going to use a wallet called Electrum.

We’re using electrum for a few reasons. It works on almost any computer. It’s fast and lightweight. And it includes all the functionality we’ll need.

To use electrum, you’ll need to install it on your computer. Let’s work through the details of how to do that.
