05. Wallet Types

Wallet Types

Previously, you learned the difference between deterministic and non-deterministic wallets. To understand each of them a little better, it helps to focus on each of them specifically and discuss them in a little more detail.

In this section, we'll do a deep dive on these 2 main types of wallets. This will inform decisions you make later on about the wallets you'll use personally or when building your own applications.

Wallet Scenarios

Coming up we made up a variety of scenarios for you to explore. Based on what you know, consider the situation and decide which type of wallet might be most suitable.

Scenario 1

I want to create paper wallets derived from one master key so I can store and recover all of them deterministically. Which wallet is used to achieve this?

SOLUTION: Sequential Deterministic Wallet

Scenario 2

I’m creating a web server that sells widgets and want to generate an unique transaction address for every customer and every address and track it independently. I want to be able to take a master public key and generate a sequence of sub-public keys, each associated with a transaction and put it on a public web server and make sure the web server has no private keys.

SOLUTION: Hierarchical Deterministic Wallet

Scenario 3

A shoe supply chain is being audited. The auditors are given a public key so they can see all the transactions of the sub tree but they can’t unlock it. Which wallet is used to achieve this?

SOLUTION: Hierarchical Deterministic Wallet

Scenario 4

A blockchain social media website uses private keys to secure personal user data. It uses wallets for backend services that use private keys that are not derived from a seed. Which wallet is used to achieve this?

SOLUTION: Non-Deterministic Wallet

Wrap Up

Throughout this section, we looked closer at the differences between deterministic and non-deterministic wallets. Then, you went through a variety of scenarios to help determine which type of wallet is best suited for a given circumstance.
