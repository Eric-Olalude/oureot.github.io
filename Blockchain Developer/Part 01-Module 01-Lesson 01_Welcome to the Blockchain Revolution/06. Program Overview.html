06. Program Overview

Program Overview

Take a look at the course syllabus for more information about the content and projects you'll complete throughout the Nanodegree.

Program prerequisites

Now you are starting your journey towards becoming a Blockchain Developer, and we would like to provide you the best learning experience during this journey.

For your best learning experience, it is important that you clearly understand the programming languages, tools, and technologies we are going to use during this program.

Prerequisites

In order to successfully finish projects and exercises in this program, you should have programming experience with JavaScript. Also, the Node.js framework would be extensively used in this course.

  • You should have JavaScript programming experience to be successful in this program.

  • ** You are required to use Asynchronous programming in JavaScript in this program. **

Specifically, you will use the Promises API or callbacks. In this program, we provide you with good resources to learn about asynchronous programming, but it is important you understand that we can’t provide a full course on this topic.

Why Asynchronous programming?

“An asynchronous model allows multiple things to happen at the same time. When you start an action, your program continues to run. When the action finishes, the program is informed and gets access to the result (for example, the data read from disk).”

From https://eloquentjavascript.net/11_async.html

This is a very important topic to know. Developers are continuously looking for ways to improve their applications with better response times, less computational resources, and low memory usage, etc. Asynchronous programming helps achieve a lot of these objectives.

In fact, most of the NPM modules you will need during this course are implemented using Promises.

  • Use of ES6 classes (Not required)

“JavaScript classes, introduced in ECMAScript 2015, are primarily syntactical sugar over JavaScript's existing prototype-based inheritance. The class syntax does not introduce a new object-oriented inheritance model to JavaScript.”

From https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Classes

This is not a an actual prerequisite for this program as its definition suggests. But it is always good to introduce better practices, so you will find some pieces of code or exercises that use ES6 classes.

  • Node.js Web Framework (Not required)

You will use the Node.js Web Framework to create RESTful APIs during this program, and we provide a few helpful resources to get you started with this.
