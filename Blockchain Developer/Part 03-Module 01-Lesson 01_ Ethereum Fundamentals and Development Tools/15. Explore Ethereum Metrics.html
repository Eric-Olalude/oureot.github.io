15. Explore Ethereum Metrics

Explore Ethereum Metrics

Now that you have some background in Ethereum, we'll take a look at Ethereum blockchain metrics.

This is a great way to monitor important metrics about the Ethereum blockchain as you developing applications. Using these metrics, you'll also gain a better understanding of metrics that you can read into your own applications.

Objectives

Our goal is to help you do the following things.

  1. Explain a variety of network statistics relating to it's performance.
  2. Utilize the web3 library documentation to search basic Ethereum statistics from your local environment.

3. Use Network Statistics

Knowing the range of network statistics, you can now start utilizing some of them for yourself using the web3 library. Try using the techniques described earlier to bring this data into your own environment. Most of this can be found through searching the documentation, but here are some examples to help get you started.

Task Description:

Try running each of the commands below to get a variety of network statistics using web3.

Example 1: getGasPrice found here

web3.eth.getGasPrice([callback])

Example 2: Get Uncle found here

eb3.eth.getUncle(blockHashOrBlockNumber, uncleIndex [, returnTransactionObjects] [, callback])

Example 3: Get Block Transaction Count found here

web3.eth.getBlockTransactionCount(blockHashOrBlockNumber [, callback])
