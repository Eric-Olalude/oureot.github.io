Get Testnet Coins

Note: The current version for Bitcoin Core is 0.17.0.1, some functions and commands has been changed since the video was recorded, follow the instructions below

Bitcoin Core version v0.17.0.1 Instructions

Change 1. Creating receiving addresses

When you open the Receiving Addresses option from your menu you will find that there isn't a "New" button option to create a new receiving address:

To create receiving address you will have to do the following steps:

  • Open the Debug Window in the Menu -> Help -> Debug Window.
  • In the Debug Window open the Console tab.
  • Type in the command line getnewaddress optionally you can add a label: getnewaddress test

This automatically creates a new receiving address in your wallet:
