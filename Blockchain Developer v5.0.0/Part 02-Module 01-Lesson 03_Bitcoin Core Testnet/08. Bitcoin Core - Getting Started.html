08. Bitcoin Core - Getting Started

Bitcoin Core Version

We made this lesson using Bitcoin Core version 0.16. In the following video, we walk you through how to download the latest release of the Bitcoin Core wallet at the time.

As this space matures, new versions are released. We wanted to provide the link to access a similar version we used while making this lesson and also provide text notes in case you want to use the latest version.

Use the following link to download: https://bitcoincore.org/bin/bitcoin-core-0.16.3/. But if you want to use the latest version we include text notes to help you in the transition.

Notes

You can using pruning to reduce the amount of space taken by the bitcoin blockchain.

  • Step 1: Create a bitcoin.conf file in your blockchain data directory. Find where Bitcoin creates its default data directory depending on your OS here.
  • Step 2: In the bitcoin.conf file, write prune=550 and save file.

This will not make blockchain sync faster. It will make it take less space, though, which can be helpful.
