06. Scan the Ethereum Network

Scan the Ethereum Network

It's time to get your first look at the Ethereum network!

To do this, you'll use one of the tools that was introduced in the previous concept, Etherscan. We'll use this tool to help explain the existing Ethereum networks, search for accounts, and find transactions on the main network.

Objectives

Our goal is to help you do the following things.

  1. Explore and describe the different Ethereum networks using Etherscan.
  2. Find large accounts on the main network and explore the details of their transactions.
  3. Find existing smart contracts that have been deployed to the Ethereum Network.

1-3. Explore Ethereum Networks Using Etherscan

To follow along with the upcoming video, go to etherscan.io. On this site, you'll walk navigate data across different Ethereum blockchain networks. All of objectives 1-3 are covered in the following video.

6.1-3-Explore-Ethereum-Networks-Using-Etherscan

1. Ethereum Networks

Which Ethereum networks can you explore using Etherscan?
SOLUTION:
  • Ethereum Main Network
  • Ropsten Test Network
  • Rinkeby Test Network

2. Ethereum Accounts

Which of the following can't be seen from the overview of an account on Etherscan?
SOLUTION: Gas Cost

3. Ethereum Contracts

Which of the following can be seen in the overview of a contract account on Etherscan?
SOLUTION:
  • Balance
  • Ether Value
  • Transactions
  • Contract Code
  • Address Watch
  • Contract Creator
  • Token Tracker
